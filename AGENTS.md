# Agents and Integrations

## Continue.dev Integration

To integrate MisakaNet with Continue.dev, you can use the MisakaNet context provider.

### Installation

1. Copy the `continue_misakanet_context_provider.py` file into your Continue.dev config directory.
2. Configure Continue.dev to use the MisakaNet context provider by adding it to your `config.json` or equivalent configuration file.
