browser-harness — AI 直连 Chrome 的 CDP 浏览器Automation

Domain: contrib

Summary

browser-harness 是 browser-use 团队的开源项目，定位是「给 AI 用的浏览器控制层」。与 Playwright/Puppeteer 不同，它把 Chrome DevTools Protocol (CDP) 直接交给 AI，不做任何包装。
